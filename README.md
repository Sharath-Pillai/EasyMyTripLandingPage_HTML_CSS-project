# EasyMyTrip Landing Page Clone
A responsive landing page clone of the EasyMyTrip.com travel booking website.[it was my initial phase project when i was learning HTML and CSS but i did not used git that time so i posting now.This project teach me more HTML and CSS features.]


## Project Overview

This project is a replica of the EasyMyTrip landing page featuring:

- Modern, responsive design
- Navigation header with service options (EMTMate, EMTDesk, EMTRoyal, Explore Bharat)
- Travel booking interface
- Mobile-friendly layout

## Files Structure

```
EasyMyTrip-Landing_Page_Clone/
├── index_Easytrip.html    # Main HTML file
├── style_Easytrip.css     # Styling
├── images/                # Image assets (logo, icons, etc.)
└── README.md             # This file
```

## Technologies Used

- **HTML5** - Markup structure
- **CSS3** - Styling and responsive design
- **Font Awesome** - Icons library
- **Google Fonts (Poppins)** - Typography

## How to Use

1. Clone or download this repository
2. Open `index_Easytrip.html` in your web browser
3. View the landing page and explore the design

## Features

- Responsive design for desktop and mobile devices
- Clean and modern UI
- Navigation menu with service offerings
- Travel search interface mockup
